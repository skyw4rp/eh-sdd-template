# PROJECT RULES

Este proyecto utiliza EH-SDD.

Reglas:

1. No implementar nada que no exista en SPEC.md.
2. No modificar arquitectura sin actualizar DESIGN.md.
3. Toda tarea debe existir en TASKS.md.
4. Toda implementación debe actualizar IMPLEMENTATION_LOG.md.
5. Toda sesión debe actualizar PROJECT_STATUS.md.
6. Nuevas ideas deben ir a BACKLOG.md.
7. AI_CONTEXT.md tiene prioridad sobre cualquier conversación.
